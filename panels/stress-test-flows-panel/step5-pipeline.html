üìã Scaling Decision Guidelines

  • Code Node: Set maxReplicas based on expected concurrent image streams. Default: 8 replicas handles moderate load.
  • ResNet Service: Configure maxReplicas to match classification capacity needs. Consider batch size and processing time.
  • Load Testing: Start with lower maxReplicas and increase based on observed queue lengths and processing times.
  • Resource Limits: Ensure cluster has sufficient resources (CPU/memory/GPU) to support configured maxReplicas.
