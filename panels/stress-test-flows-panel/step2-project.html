Create Project

Create a project in your organization for stress testing. The project will contain datasets and pipelines.

Create faasProxy Dataset

Create a dataset with the faasProxy storage driver. This driver allows the dataset to reference files stored in the FaaS Filestore without copying them, using link items.

Retrieving Storage Driver ID: To get the storage driver ID for faasProxy type in your organization, use the following approach:

import dtlpy as dl import requests # Login dl.login() # Get your organization org = dl.organizations.get(organization_id='YOUR_ORG_ID') # Get storage drivers for the organization headers = {'Authorization': f'Bearer {dl.token()}'} base_url = dl.environment() # List storage drivers response = requests.get( f"{base_url}/organizations/{org.id}/storage-drivers", headers=headers ) if response.status_code == 200: drivers = response.json() # Find faasProxy driver faas_proxy_driver = None for driver in drivers: if driver.get('type') == 'faasProxy' or driver.get('driverType') == 'faasProxy': faas_proxy_driver = driver break if faas_proxy_driver: driver_id = faas_proxy_driver.get('id') or faas_proxy_driver.get('driverId') print(f"Found faasProxy driver ID: {driver_id}") print(f"Driver details: {faas_proxy_driver}") else: print("faasProxy driver not found in organization") else: print(f"Failed to get storage drivers: {response.status_code} - {response.text}") # Use the driver_id when creating dataset # dataset = project.datasets.create( # dataset_name='benchmark-images', # driver_id=driver_id # Use the retrieved driver ID # )
